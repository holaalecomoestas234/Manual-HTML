Lista de etiquetas de Sublime Text (Primera Parte)


Etiquetas de formato de texto
Para colocar textos en Negrita debes escribir el texto < b > (Texto que deseas en Negrita) y < /b > al finalizar

Si deseas escribir texto con cursiva debes escribir < i > (Texto que deseas en cursiva) y < / i >

Para colocar textos subrayados debes escribir el texto < u > (Texto que deseas subrayado) y < /u >

Si tienes que colocar saltos de linea se debe usar la etiqueta < br > al final de cada texto donde desees bajar o un salto de linea
Al desear marcar un texto diferente puedes utilizar el < mark > para marcarlo de un fondo amarillo
Buscando como tener un texto mas pequeño puedes utilizar el < small > para cear un texto de menor tamaño
Para conseguir un texto tachado deberás usar < del > para conseguir un texto que luzca tachado
Para conseguir tamaños de textos diferentes usarás < font size="Numero que desees" > y así podrás elegir el tamaño de texto de tu agrado
Regresando con los textos más importantes puedes usar < strong > para conseguir un texto que se vea más importante que los otros
Pero si buscas un texto con enfasis debes usar < em > y así conseguirás un texto con este estilo
